Score each dimension 0, 1, or 2 against the submission (a short data report
written under hard format constraints). Formatting compliance is graded
elsewhere — you are scoring the *depth qualities* of the prose.

## quantification
Are comparisons carried by numbers computed from the data (percentages,
ratios, gaps) rather than bare rankings ("X leads", "Y trails")?
0 = rankings only; 1 = some comparisons quantified; 2 = essentially every
comparison carries a computed figure.

## specificity
Are the recommendations tied to named products, regions, and figures from the
data — actionable as written — rather than generic advice that would fit any
sales report?
0 = generic; 1 = mixed; 2 = every recommendation is specific and data-grounded.

## insight_density
Within the word cap, does the prose surface non-obvious, decision-relevant
observations (concentration, gaps, outliers) rather than restating totals the
tables already imply?
0 = filler/restatement; 1 = one real observation; 2 = multiple observations a
reader could act on.
